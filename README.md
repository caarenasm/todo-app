Proyecto Frontend en Ionic

Este proyecto es una aplicación desarrollada con **Ionic + Angular**, pensada para ejecutarse en navegadores y dispositivos móviles.

## Requisitos previos

Antes de comenzar, asegúrate de tener instaladas las siguientes herramientas:

Herramienta | Versión recomendada | Verificación |
| **Node.js** | 18.x o superior | `node -v` |
| **npm** | 9.x o superior | `npm -v` |
| **Ionic CLI** | Última versión estable | `npm install -g @ionic/cli` |

## Si no tienes instalado Ionic, puedes hacerlo con:

npm install -g @ionic/cli

## Clonar el proyecto, Ejecuta los siguientes comandos en tu terminal:
git clone git clone https://github.com/caarenasm/todo-app.git
cd todo-app

## Después de clonado y abierto en el editor para levantar ejecute el comando para actualizar antes de.
npm install

## Luego se ejecuta el proyecto con el siguiente comando
ionic serve
